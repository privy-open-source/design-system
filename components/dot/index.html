Dot #

Dot is a sub-component used for Badge and Label

Usage #

Basic Usage #

preview
vue
<template>
  <p-dot />
</template>

Colors #

Dot available in 7 different colors primary, secondary, success, info, warning, danger, gold. default is primary

preview
vue
<template>
  <p-dot color="primary" />
  <p-dot color="secondary" />
  <p-dot color="success" />
  <p-dot color="info" />
  <p-dot color="warning" />
  <p-dot color="danger" />
  <p-dot color="gold" />
</template>
